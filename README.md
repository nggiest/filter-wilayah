Frontend Assesment - Filter Wilayah Indonesia

Tech Stack :
React 18, Typescript, MobX, Tailwind CSS, Font Awesome

Struktur Data :
Aplikasi ini menggunakan data JSON lokal yang terletak di:
/public/data/indonesia_province.json
/public/data/indonesia_city.json
/public/data/indonesia_distric.json

Cara menjalankan project :

1. Clone Repository ini di https://github.com/nggiest/filter-wilayah
2. Masuk ke folder project di cd filter-province
3. Install dependensi : npm install
4. Jalankan : npm run dev

Live demo :
https://filter-wilayah-ggg.vercel.app/
